# Confluence Sync

When making changes to `scripts/provision.yml`, update the Confluence page [Gaming PC — Ansible Provisioning](https://smithsimms.atlassian.net/wiki/spaces/HA/pages/113573889/Gaming+PC+Ansible+Provisioning) in the Home Ops space to reflect those changes. This includes:

- New or removed software packages
- Changes to Windows Update configuration
- Changes to SSH, firewall, or registry settings
- Any new tasks or removed tasks
